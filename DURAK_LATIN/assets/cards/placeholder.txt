Для тестирования игры необходимо создать следующие изображения карт:

1. back.png - рубашка карты

2. Карты по мастям:
   - hearts_6.png, hearts_7.png, ..., hearts_ace.png
   - diamonds_6.png, diamonds_7.png, ..., diamonds_ace.png
   - clubs_6.png, clubs_7.png, ..., clubs_ace.png
   - spades_6.png, spades_7.png, ..., spades_ace.png

Временно можно использовать любые изображения с соответствующими именами файлов.
Размер изображений должен быть 256x384 пикселя.

Для создания простых карт можно использовать любой графический редактор или онлайн-сервис.

Пример создания простой карты:
1. Создайте белый прямоугольник размером 256x384 пикселя
2. Добавьте символ масти в верхнем левом и нижнем правом углах
3. Добавьте номинал карты в центре
4. Сохраните файл с соответствующим именем (например, hearts_6.png)
